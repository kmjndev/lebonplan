Note

🔑 Google OAuth

Pour activer la connexion Google, créez des identifiants OAuth 2.0 dans Google Cloud Console.

Étapes :
1. console.cloud.google.com → créer un projet
2. APIs & Services → Identifiants → OAuth 2.0
3. Type : Application Web
4. Origines JS : http://localhost ou votre domaine
5. Copiez le Client ID ci-dessous

⚠ Fonctionne uniquement sur HTTP/HTTPS. Pas en file://. Utilisez python3 -m http.server 8080 ou déployez en ligne.
